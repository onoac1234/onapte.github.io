React journey - Part 1

So, I have finally started my React journey. The first basic part to learn in React is about components. Understanding why we need components can take a little bit of time, but it is worth the effort.


I first began with a simple book app where the idea is to create a book store like interface where a number of different books are displayed with their details. So, it should look something like below.
Please disregard the "Google" symbol. Ofcourse, I would like to add some more details for each of the book.


So, I opened up the command prompt (I am on Windows) and created an app named book-store using create-react-app.
The initial interface will look like this (inside VSCode).
I quickly created a new component named "Book" and imported it to the App.js file. The code which I wrote inside the Book.js file is given below. 

import React from 'react'

function Book(props) {
    return (
        <div className="book"<
        <img src="https://kbimages1-a.akamaihd.net/219159f1-5a27-42a0-a3d7-6c3df5a521b5/1200/1200/False/treasure-island-78.jpg" className="book-image" /> <br />
            Title: { props.title } <br />
            Price: ${ props.price } <br />
            Author: { props.author } <br />
        </div>
    )
}

export default Book;

And the corresponding CSS below. 

/* css for Book */
.book {
  background-color: black;
  color: white;
  font-size: 20px;
  font-family: 'Courier New', Courier, monospace;
  text-align: center;
  margin: 0 auto;
  padding: 10px;
  max-width: 800px;
  width: 75%;
  border: 1px solid ghostwhite;
  margin-top: 15px;
  margin-bottom: 15px;
}

.book-image {
  height: 200px;
}
